# 랜딩페이지 와이어프레임

> 참고 디자인(Canva 시안)의 **화면 구성**만 정리한 메모입니다.
> 디자인/색/글꼴 스펙은 별도 문서로 만들지 않고, 빌드 시 코드 한 곳(CSS 변수)에 모읍니다.
> 콘텐츠(문구·로고·이미지)는 추후 **인카금융서비스 프로사업단총괄 성수본부** 내용으로 교체 예정.

---

## 전체 톤 (시안 기준)
- 분위기: 다크 베이스(검정) + 보라→주황 그라데이션 포인트
- 섹션이 **검정 배경 ↔ 그라데이션 배경**으로 번갈아 가며 리듬을 만듦
- 둥근 모서리 카드, 큼직한 헤드라인, 3D 그라데이션 도형 장식

---

## 화면 구성 (위 → 아래)

### 1. 헤더 / 내비게이션 (상단 고정)
- 좌측: 로고/브랜드명
- 우측: 메뉴 + CTA 버튼
- (시안엔 약하게 표현 — 빌드 시 sticky 헤더로 구성)

### 2. 히어로 (Hero)
- 배경: 검정
- 상단 작은 배지(pill): `※ 브랜드명`
- 큰 헤드라인 (그라데이션 강조 텍스트 포함)
  - 예: "역사에 한 획을 긋는 / 브랜드가 될 수 있게 / 도와드립니다"
- CTA 버튼 1개: "둘러보기"

### 3. 소개 / 가치 (2-컬럼 카드)
- 배경: 보라→주황 그라데이션
- 섹션 타이틀
- 카드 2개 (이미지 + 제목 + 부제)
  - 카드: 대표 이미지, 굵은 제목, 한 줄 설명

### 4. 작업/항목 그리드 (3-컬럼 카드)
- 배경: 검정
- 카드 3개 (이미지 + 제목 + 분류)
  - 예: 콘텐츠 개발 / 그래픽 디자인 / 웹 디자인

### 5. 피처 블록 A (이미지 + 텍스트)
- 배경: 검정
- 좌: 큰 이미지(둥근 카드) / 우: 제목 + 설명 문단
- 제목 예: "○○○ 사진 촬영"

### 6. 미디어 / 영상 블록
- 가로 풀폭 이미지 + 가운데 ▶ 재생 버튼 (영상 임베드 자리)
- 캡션/제목 오버레이

### 7. 피처 블록 B (텍스트 + 장식 도형)
- 배경: 검정
- 좌: 제목 + 설명 문단 (예: "1981년 설립" 같은 연혁/스토리)
- 우: 3D 그라데이션 블롭 장식

### 8. 파트너 / 로고 그리드
- 배경: 보라→주황 그라데이션
- 로고(또는 텍스트 로고) 6개 내외, 2~3열 그리드

### 9. 마감 CTA / 연락 섹션
- 배경: 컬러풀한 이미지 + 오버레이
- 배지(pill) + 큰 문구: "여기서 여러분의 이야기를 시작해보세요"
- 연락처(이메일/전화) 또는 CTA 버튼

### 10. 푸터 (Footer)
- 배경: 검정
- "소셜 미디어" + SNS 아이콘 (facebook / twitter / instagram)
- (빌드 시 사업자 정보·연락처 추가 가능)

---

## 빌드 메모
- 색상/글꼴은 `:root` CSS 변수로 한곳에 모아 한 번에 교체 가능하게 한다.
- 섹션 순서는 위 1~10을 기본으로 하되, 비즈니스 콘텐츠 확정 후
  필요 섹션(상담신청 폼, 수수료/급여 테이블, 후기, FAQ 등)을 추가/교체한다.
